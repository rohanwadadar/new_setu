# 🛡️ Data Privacy & Non-Training Rule

## Objective
Ensure that no data from this workspace is used for training the Antigravity model or any other large language models.

## Instructions
1. **Strict Data Isolation**: Do not upload, send, or use any content from this project to improve internal AI models.
2. **Context Restriction**: Respect the `.geminiignore` file at all times. Files listed there must never be read, indexed, or used as part of any reasoning process.
3. **Telemetry Awareness**: Always operate as if telemetry is disabled, even if the system-level setting is unclear. Prioritize the user's privacy above all else.
4. **No Learning**: Do not attempt to "learn" from patterns in this codebase for use in other projects or for model improvement. This interaction is for execution within this local environment only.

## Scope
This rule applies to all interactions within the `rohanwadadar/new_setu` workspace.
